# 💰 Finance Tracker API (Week 12 Submission)

## 📌 Project Overview
This is a simple RESTful API built using **Node.js and Express** for managing financial transactions.  
It allows users to add, view, and delete transactions using HTTP requests.

This project is part of the Week 12 Full-Stack Development Final Submission.

## 🚀 Features
- Add new transactions
- Retrieve all transactions
- Delete transactions by ID
- RESTful API structure
- CORS enabled for frontend integration
- JSON-based responses

---

## 🛠 Tech Stack
- Node.js
- Express.js
- JavaScript (ES6)
- CORS middleware
